SE–X-ray Hybrid Metrology Conference Presentation

From coursework analysis to conference feedback

수업 프로젝트에서 시작한 문헌 기반 계측 분석을 특별세션에서 공동 발표하고, 현장 질의응답을 통해 3D 패턴 적용과 계면 아래 다층 구조까지 후속 검토 범위를 구체화한 경험입니다.

01 / PRESENTATION

Presenting the metrology strategy.

수업 프로젝트에서 시작한 SE–X-ray Hybrid Metrology 분석을 외부 피드백에 따라 보완한 뒤, 2026년 반도체공학회 하계종합학술대회 특별세션에서 공동 발표했습니다.

발표에서는 SE 단독 역모델링에서 발생할 수 있는 두께·광학상수·계면 거칠기 간 파라미터 상관성을 설명하고, XRR을 중심으로 한 X-ray 구조정보를 SE 모델의 고정값 또는 제약조건으로 활용하는 feed-forward hybrid metrology 전략을 제시했습니다.

계측 원리와 프로젝트 발전 과정에 대한 자세한 분석은 기술 프로젝트 페이지에서 확인할 수 있습니다.

02 / FEEDBACK

Questions that expanded the analysis.

POSITIVE FEEDBACK

Manufacturing applicability

3D 패턴 대응, SE와 X-ray의 측정 영역 차이, 그리고 양산 적용 가능성을 추가로 검토한 점이 프로젝트의 현실성을 높였다는 피드백을 받았습니다.

QUESTION RECEIVED

Interaction below the interface

현재 분석이 High-k 박막과 계면 특성에 집중되어 있는데, 계면 아래의 기판 또는 인접층과의 상호작용으로 나타나는 다른 측정 결과까지 고려했는가라는 질문을 받았습니다.

03 / FOLLOW-UP

Extending the model below the interface.

질의응답을 통해 계면을 독립적인 경계로만 해석하는 데 한계가 있음을 확인했습니다. 향후에는 High-k 박막, 계면층과 하부 기판이 연결된 다층 구조에서 나타나는 광학적·구조적 응답을 함께 검토하고자 합니다.

  • 하부 기판과 계면층 물성이 SE 광학 신호에 미치는 영향
  • 층간 밀도와 조성 변화가 XRR 신호에 미치는 영향
  • 하부 구조를 포함할 때 발생하는 모델 파라미터 상관관계
  • 계면 단독 모델과 다층 상호작용 모델의 결과 차이
  • 각 계측법의 측정 영역과 불확실성이 constraint에 미치는 영향
Follow-up scope: 위 항목은 완료된 실험 결과가 아니라 학회 피드백을 통해 설정한 후속 분석 방향입니다.
