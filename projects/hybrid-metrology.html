02 / METROLOGY CHALLENGE

Why one fitted structure is not enough.

ALD 초기 성장에서는 국소적인 아일랜드 성장과 미세 기공, 수직 밀도 구배, High-k와 기판 사이의 계면층이 함께 나타날 수 있습니다. 초박막에서는 thickness, n·k, density, roughness 변화가 서로 유사한 측정 응답을 만들 수 있어 하나의 낮은 MSE가 유일한 물리적 해를 보장하지 않습니다.

ALD interface problem

초기 성장과 계면층을 단순한 하나의 평면 경계로만 모델링할 때 발생할 수 있는 구조적 모호성을 정리했습니다.

PARAMETER COUPLING

Correlated thin-film properties

두께·광학상수·밀도·거칠기 변화가 서로 보상하면 비슷한 광학 응답을 만들 수 있습니다.

PHYSICAL VALIDITY

Low MSE ≠ unique structure

수학적 fitting 오차가 낮아도 실제 박막 구조와 가장 가까운 해라고 단정할 수 없습니다.

03 / STANDALONE SE

Inverse modeling and parameter correlation.

SE는 측정한 Ψ와 Δ를 가상 optical model에 맞춰 두께와 n·k 등을 역추정합니다. 초박막에서는 광학적 위상 변화가 작아져 감도가 감소하고, 선택한 모델과 초기 조건에 따라 다른 구조가 유사한 결과를 만들 수 있습니다.

SE parameter correlation

최적해가 한 점에 수렴하지 않고 길게 늘어진 confidence region으로 나타날 수 있는 이유를 분석했습니다.

04 / X-RAY METROLOGY

Structural information from XRR.

High-k 초박막에서는 XRR을 핵심 X-ray 계측법으로 다뤘습니다. 낮은 입사각에서 얻은 반사 강도의 특징을 통해 SE와 다른 물리 원리에 기반한 구조정보를 제공합니다.

XRR measurement principle

Critical angle, Kiessig fringe, reflectivity decay가 각각 밀도·두께·거칠기 정보와 연결됩니다.

XRR featureRelated informationRole in hybrid analysis
Critical angleElectron density · film density밀도 파라미터의 구조 제약 후보
Kiessig fringeThin-film thickness두께 초기값 또는 허용 범위
Reflectivity decaySurface · interface roughness거칠기 모델의 물리적 검토
XRD / XRF literatureCrystal · composition information구조 제약의 확장 문헌 사례

05 / FEED-FORWARD STRATEGY

Reducing free parameters with independent constraints.

핵심 전략은 X-ray 계측에서 얻은 구조정보를 SE 모델의 초기값·고정값·허용 범위로 먼저 전달하는 것입니다. 자유 파라미터 수를 줄인 뒤 SE를 다시 fitting하고, 수학적 오차와 물리적 일관성을 함께 확인합니다.

Feed-forward hybrid workflow

독립 계측정보를 이용해 SE 역모델링의 파라미터 탐색 범위를 줄이는 의사결정 흐름입니다.

  1. X-ray measurement

    목표 구조와 시료에 맞는 X-ray 정보를 확보합니다.

  2. Structural extraction

    두께·밀도·거칠기 등 독립 구조정보를 해석합니다.

  3. SE model constraints

    구조정보를 고정값 또는 허용 범위로 변환합니다.

  4. Reduced free parameters

    동시에 추정해야 하는 모델 변수의 수를 줄입니다.

  5. Refitting

    제약된 모델로 Ψ·Δ 데이터를 다시 fitting합니다.

  6. Consistency check

    MSE와 X-ray 구조정보의 물리적 일관성을 함께 검토합니다.

Technical judgment: 잘못된 constraint는 SE 모델에 bias를 만들 수 있으므로, 고정값 사용 자체가 아니라 계측 불확실성과 모델 타당성을 함께 검토하는 것이 중요합니다.

06 / 3D & MANUFACTURING

Applicability depends on measurement scale.

Blanket wafer의 평면 다층 모델을 patterned wafer에 그대로 적용하기는 어렵습니다. SE와 X-ray의 spot size 및 측정 영역이 다를 수 있으므로 같은 위치·같은 구조를 대표하는지 먼저 확인해야 합니다.

PATTERNED WAFER

OCD / MMSE modeling

3D 패턴에서는 높이·폭·sidewall angle을 포함하는 광학 형상 모델과 Mueller Matrix 기반 접근 가능성을 검토했습니다.

LOCAL X-RAY

Micro-Spot measurement

측정 영역 mismatch를 줄이기 위해 작은 X-ray beam으로 동일 패턴 영역을 비교하는 방안을 조사했습니다.

  • SE와 X-ray의 spot size 및 sampling area 차이 확인
  • Blanket wafer와 patterned wafer의 분석 전략 분리
  • 빠른 optical mapping 후 대표·이상 지점에 X-ray를 적용하는 sampling 전략 검토
  • 양산 적용 시 throughput, spot size, repeatability와 장비 간 정합성 고려
Feasibility scope: 위 내용은 직접 Fab에서 검증한 양산 결과가 아니라 문헌과 피드백을 바탕으로 정리한 적용 가능성 검토입니다.

07 / LIMITATIONS

What this analysis does not prove.

  • 문헌 기반 분석이므로 직접 측정한 SE/XRR dataset이나 새로 생성한 계측 데이터가 없습니다.
  • XRR 역시 model dependency가 있으며 서로 다른 다층 구조가 유사한 반사율 곡선을 만들 수 있습니다.
  • 부정확한 constraint는 자유도만 줄이고 SE 모델에 bias를 만들 수 있습니다.
  • 복잡한 패턴 구조에서는 평면 multilayer model의 적용 범위가 제한됩니다.
  • 각 계측법의 측정 불확실성과 공간적 대표성을 함께 고려해야 합니다.
