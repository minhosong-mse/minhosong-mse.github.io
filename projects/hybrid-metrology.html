THIN-FILM METROLOGY · CONFERENCE PRESENTATION

SE–XRR Hybrid Metrology

ALD-based High-k Thin Films

ALD High-k 초박막에서 standalone spectroscopic ellipsometry의 파라미터 상관성 문제를 검토하고, XRR 구조 정보를 제약조건으로 결합해 fitted solution의 물리적 신뢰성을 높이는 논리를 발표했습니다.

DATE
2026.07.15
STATUS
Presented
ROLE
Presenter · Hybrid-metrology logic analysis
METHODS
Spectroscopic Ellipsometry, XRR, literature review

01 / OVERVIEW

Why low fitting error is not enough.

SE는 비파괴·고속 광학 계측에 유용하지만, ALD 기반 High-k 초박막에서는 두께, n·k, roughness 같은 파라미터가 서로 강하게 상관될 수 있습니다. 이때 낮은 fitting error를 갖는 여러 해가 존재해도 모두 물리적으로 타당하다고 볼 수 없습니다.

Core question: 독립 계측으로 얻은 구조 정보를 SE 모델에 제약조건으로 적용하면 해의 비유일성과 비물리적 수렴을 줄일 수 있는가?

02 / LOGIC

Independent constraints for a correlated model.

  1. Standalone SE

    광학 모델에서 thickness, n·k, roughness가 함께 변할 때의 상관성 한계를 검토했습니다.

  2. XRR structure

    XRR에서 얻는 두께·밀도·계면 정보를 독립적인 물리 구조 정보로 다뤘습니다.

  3. Hybrid fitting

    XRR 값을 fixed value 또는 constraint로 적용해 SE 해의 물리적 범위를 좁히는 접근을 정리했습니다.

My contribution

  • standalone SE의 parameter correlation과 비유일 해 문제를 발표 논리로 구조화했습니다.
  • XRR 구조 제약이 fitted solution의 물리적 신뢰성에 기여하는 방식을 문헌 기반으로 정리했습니다.
  • 학회 특별세션 발표 자료를 구성하고 발표를 수행했습니다.

03 / INSIGHT

Physical reliability through hybrid metrology.

핵심 결론은 fitting error가 낮다는 사실만으로 모델 파라미터의 물리적 타당성이 보장되지 않는다는 점입니다. 서로 다른 원리에 기반한 XRR 정보를 결합하면 SE 역문제의 자유도를 제한하고 해석의 물리적 신뢰성을 개선할 수 있습니다.

Limitations & lessons

  • 본 작업은 hybrid-metrology 논리와 문헌 분석을 다룬 발표이며, 새로운 실험 데이터 취득 결과를 주장하지 않습니다.
  • XRR 역시 모델과 측정 조건의 영향을 받으므로 제약조건의 불확실성을 함께 고려해야 합니다.
