THIN-FILM METROLOGY · 2026 ISE CONFERENCE PRESENTATION

SE-X-ray Hybrid Metrology

ALD 기반 High-k 박막의 미세 밀도 및 계면 거칠기 정량화

ALD High-k 초박막 분석에서 standalone spectroscopic ellipsometry가 갖는 파라미터 상관성과 비유일 해 문제를 검토하고, X-ray 계측으로 확보한 독립 구조 정보를 SE 모델의 고정값 또는 제약조건으로 적용하는 hybrid-metrology 접근을 발표했습니다.

01 / OVERVIEW

Why low fitting error is not enough.

이 발표는 ALD 기반 High-k 초박막의 밀도와 계면 거칠기를 정량화할 때 standalone SE inverse modeling이 갖는 한계를 검토하고, 서로 다른 계측 원리에서 얻은 정보를 결합하는 접근의 필요성을 정리했습니다.

  • 박막 두께, 광학상수 n·k, 계면 거칠기 사이의 파라미터 상관성을 분석했습니다.
  • 서로 다른 물리 구조가 유사한 광학 신호를 만들 수 있는 비유일 해 문제를 검토했습니다.
  • 낮은 fitting error와 물리적으로 타당한 해가 반드시 동일하지 않을 수 있음을 문헌 사례로 비교했습니다.
Core question: X-ray 계측으로 얻은 독립적인 구조 정보를 SE 모델에 제약조건으로 적용하면 파라미터 모호성과 비물리적 수렴을 줄일 수 있는가?

02 / METHOD

Feed-forward constraints for a correlated model.

  1. Standalone SE modeling

    thickness, optical constants n·k, interface roughness를 함께 조정하는 과정에서 발생하는 parameter correlation을 검토했습니다.

  2. X-ray structural information

    XRD/XRR에서 확보할 수 있는 구조적·물리적 정보와, 문헌이 뒷받침하는 thickness, density, interface 관련 제약을 정리했습니다.

  3. Feed-forward constraint

    X-ray-derived 정보를 fixed value 또는 constraint로 SE 모델에 입력해 가능한 해의 범위와 파라미터 모호성을 줄이는 논리를 구성했습니다.

  4. Physical-validity review

    가장 작은 MSE만을 정답의 근거로 사용하지 않고 fitting quality와 물리적 타당성을 함께 비교하는 검토 기준을 적용했습니다.

My contribution

  • SE 단독 분석에서 발생하는 thickness, n·k, roughness parameter correlation 문제 분석
  • X-ray 계측 정보를 SE 모델의 fixed value 또는 constraint로 활용하는 hybrid-metrology 논리 정리
  • 문헌 사례를 통한 fitting error와 물리적 타당성의 차이 비교
  • 특별세션 발표 자료 구성 및 발표 진행

03 / INSIGHTS

Physical reliability through hybrid metrology.

Standalone SE limitation

박막 두께, 광학상수 n·k, 계면 거칠기 사이에 강한 파라미터 상관성이 발생하여 서로 다른 물리적 구조가 유사한 광학 신호를 생성할 수 있음.

Feed-forward hybrid metrology

X-ray 계측에서 확보한 구조적·물리적 정보를 SE 모델의 고정값 또는 제약조건으로 반영하여 파라미터 모호성을 줄일 수 있음.

Physical validity

단순히 MSE가 가장 낮은 모델이 실제 구조와 가장 가까운 해를 의미하지 않을 수 있으므로 fitting quality와 물리적 타당성을 함께 검토해야 함.

High-k thin-film application

서로 다른 계측 원리가 제공하는 독립적인 정보를 결합하면 ALD High-k 초박막의 밀도와 계면 거칠기를 비파괴 방식으로 정량화할 가능성이 높아짐.
