2026 ISE SUMMER CONFERENCE · SPECIAL SESSION

SE-X-ray Hybrid Metrology

ALD-based High-k thin-film density and interface-roughness quantification

분광 타원계측(SE) 단독 역모델링에서 두께, 광학상수, 계면 거칠기가 강하게 상관되는 문제를 분석하고, XRD/XRR의 구조 정보를 SE model의 제약조건으로 사용하는 feed-forward hybrid metrology 접근을 정리한 학술 발표입니다.

01 / OVERVIEW

Why one fit is not enough.

ALD 기반 High-k 초박막은 두께뿐 아니라 밀도와 계면 거칠기를 함께 알아야 공정 상태를 해석할 수 있습니다. 발표는 SE가 빠르고 비파괴적인 광학 측정이라는 장점을 가지지만, fitting error가 낮은 하나의 model만으로 물리적 구조의 유일성을 보장하기 어렵다는 문제에서 출발했습니다.

PRIMARY METHODSpectroscopic Ellipsometry
CONSTRAINTXRD / XRR structural information
TARGETALD High-k ultrathin films
OUTPUT23-slide conference presentation
발표 범위: 공개 자료는 hybrid metrology의 원리와 문헌 사례를 분석한 발표이며, 본인이 직접 수행한 High-k 실험 결과나 새로운 수치 dataset을 주장하지 않습니다.

02 / PROBLEM

Parameter correlation in standalone SE.

SE는 측정한 Ψ와 Δ를 optical model에 맞춰 두께와 n·k 등을 역추정합니다. 그러나 초박막에서는 thickness, optical constants, interface roughness가 서로 보상할 수 있어 다른 물리 구조가 유사한 optical response와 MSE를 만들 수 있습니다.

NON-UNIQUENESS

Correlated parameters

한 변수를 조정한 효과가 다른 변수의 변화로 상쇄되면 fitting minimum이 물리적으로 유일하지 않을 수 있습니다.

VALIDITY GAP

Low MSE ≠ physical truth

계산상 오차가 가장 낮은 model이 실제 density·roughness에 가장 가까운 해라는 보장은 없습니다.

03 / WORKFLOW

Feed-forward hybrid workflow.

  1. Measure with X-ray

    XRD/XRR에서 결정·밀도·두께·거칠기와 관련된 독립적인 구조 정보를 확보합니다.

  2. Define constraints

    X-ray 분석 결과를 SE model의 fixed value 또는 허용 범위로 변환합니다.

  3. Refit SE model

    자유도를 줄인 model로 SE data를 다시 fitting해 parameter ambiguity를 낮춥니다.

  4. Check physical reliability

    fitting error뿐 아니라 X-ray constraint와 일치하는지 함께 검토합니다.

04 / ANALYSIS

Connecting measurement physics.

발표는 SE의 역모델링과 covariance 문제를 먼저 설명하고, XRD의 Bragg 관계와 XRR의 critical angle·Kiessig fringe가 제공하는 독립 정보를 연결합니다. 이후 문헌의 feed-forward 사례를 통해 XRD/XRR 정보를 SE의 고정값 또는 constraint로 사용하는 논리를 단계적으로 구성했습니다.

01

SE limitation

두께–n·k–거칠기의 상관성과 error ellipse로 parameter ambiguity를 설명했습니다.

02

X-ray evidence

XRD·XRR의 측정 원리와 구조 정보가 SE와 독립적이라는 점을 정리했습니다.

03

Hybrid logic

X-ray constraint를 먼저 결정하고 SE model을 재계산하는 feed-forward 절차를 제안했습니다.

04

High-k application

ALD 초박막의 density와 interface roughness를 비파괴적으로 정량화할 가능성을 검토했습니다.

06 / CONTRIBUTION

My contribution.

  • SE 단독 계측의 parameter correlation과 낮은 fitting error의 한계를 중심 문제로 구조화했습니다.
  • XRD/XRR의 독립 정보를 SE model constraint로 연결하는 feed-forward hybrid 논리를 분석했습니다.
  • 문헌 사례와 계측 원리를 ALD 기반 High-k 초박막의 density·interface roughness 문제에 연결했습니다.

07 / LIMITS

Limitations & lessons.

  • 공개 자료는 문헌 기반 기술 발표이며 직접 수행한 SE/XRR 실험 dataset이나 새로운 정량 결과를 포함하지 않습니다.
  • XRD와 XRR은 서로 다른 정보를 제공하므로 시료 상태와 목표 parameter에 맞춰 constraint의 종류와 신뢰도를 구분해야 합니다.
  • constraint가 부정확하면 SE model의 자유도만 줄이고 bias를 만들 수 있어 각 계측의 uncertainty를 함께 다뤄야 합니다.
  • 이 프로젝트에는 source code가 없으므로 코드 뷰어를 제공하지 않고 발표 PDF와 근거 이미지를 직접 연결합니다.
