01 / OVERVIEW

A funnel, not a single filter.

신소재공학 전공 2명과 전기공학 전공 2명이 공정팀·회로팀으로 나뉘어 반도체 연구 트렌드를 분석했습니다. 논문 제목, 영문 제목, 키워드, 영문 초록을 결합한 텍스트에서 반도체 관련 문헌을 먼저 찾고, 다시 공정 관련 문헌을 좁힌 뒤 세부 분야를 분석했습니다.

02 / PROBLEM

Keywords created false confidence.

초기에는 8개 반도체 공정 범주에 폭넓은 키워드를 적용했습니다. 그러나 test, inspect, check처럼 여러 공학 분야에서 흔히 쓰이는 단어가 포함되면서 실제 반도체 공정 문헌이 아닌 논문까지 분류되는 false positive가 발생했습니다.

INITIAL APPROACH

Eight broad process categories

범용 키워드로 넓게 수집했지만 분류 수치가 실제 도메인 문헌을 과대평가했습니다.

STRATEGY PIVOT

Two domain-specific tracks

결과 검수 후 Photolithography와 Advanced Packaging으로 범위를 줄이고 전문 키워드 중심으로 재분류했습니다.

핵심 결과: false positive를 발견한 뒤 분류 전략을 변경했다는 과정 자체가 저장소와 발표자료에서 명시된 검증 결과입니다.

03 / WORKFLOW

Classification and validation workflow.

  1. Unify text fields

    NODE_TTLE, NODE_TTLE_EN, KYWD, ABST_EN을 결합해 검색 텍스트를 구성했습니다.

  2. Semiconductor filter

    전체 62,199편에서 반도체 관련 17,413편을 1차로 분류했습니다.

  3. Process filter & QA

    4,000편 이상 후보를 얻은 뒤 샘플과 키워드 빈도를 확인해 false positive를 진단했습니다.

  4. Focused analysis

    포토리소그래피와 첨단 패키징 데이터로 좁혀 연도별 추세와 핵심 기술어를 해석했습니다.

Strategy pivot

04 / ANALYSIS

Advanced packaging lens.

첨단 패키징 분석에서는 TSV, TGV, interposer, bump, chiplet, hybrid bonding, HBM, 3D stacking, warpage, reliability, fan-out을 주요 키워드로 사용했습니다. 연도별 변화와 논문 문맥을 함께 검토해 수직 집적, chiplet architecture, hybrid bonding·HBM, 열·기계 신뢰성, 검사 자동화를 핵심 흐름으로 정리했습니다.

06 / CONTRIBUTION

My contribution.

  • 팀 리더로 공정팀·회로팀의 역할, 분석 일정, 주간 산출물과 최종 발표 통합을 조율했습니다.

07 / LIMITS

Limitations & lessons.

  • 키워드 기반 분류는 문맥을 완전히 이해하지 못하므로 false positive와 false negative가 모두 발생할 수 있습니다.
  • 연도별 논문 수는 연구 관심의 proxy이며 산업 생산량, 시장 규모, 기술 성숙도를 직접 의미하지 않습니다.
  • 저장소 노트북에는 Google Drive 경로 등 원 실행 환경 의존성이 남아 있어 README의 execution note를 확인해야 합니다.
  • 이 페이지는 노트북 코드를 재게시하지 않고, 상세 재현 정보는 저장소의 notebook·CSV·weekly log로 연결합니다.
