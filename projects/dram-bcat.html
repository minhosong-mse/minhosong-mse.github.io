20 nm-Class DRAM BCAT

GIDL leakage reduction through device and process optimization

BCAT의 trench corner에 집중되는 전계와 band-to-band tunneling을 분석하고, 20 nm급 DRAM의 GIDL 누설을 줄이기 위한 소자·공정 변수 설계를 수립한 진행 중 프로젝트입니다. 현재 저장소는 연구 문제, 교수 피드백, 변수 범위와 향후 검증 계획을 문서화합니다.

01 / OVERVIEW

A leakage-focused BCAT study.

BCAT 구조는 수직 채널과 buried gate를 이용해 셀 면적을 줄이는 반면, trench 하단과 gate–drain overlap 부근의 국부 전계가 커질 수 있습니다. 연구는 이 전계가 BTBT 기반 GIDL과 refresh 특성에 미치는 영향을 공정 변수와 연결해 설명하는 것을 목표로 합니다.

FAILURE MODEGIDL / BTBT leakage
PRIMARY REGIONTrench corner & gate overlap
CURRENT EVIDENCEREADME · 8-page re-presentation deck

02 / PROBLEM

Where the electric field concentrates.

trench corner의 곡률과 gate–drain overlap은 off-state에서 높은 전계를 만들 수 있고, 이는 valence band에서 conduction band로의 tunneling 확률을 높입니다. 단순히 trench depth만 바꾸는 접근으로는 전계 분포, capacitance, 공정 실현 가능성을 함께 설명하기 어렵다는 문제가 있었습니다.

DEVICE PHYSICS

BTBT-driven leakage

Band-to-Band 모델과 380 K 조건을 포함해 off-state 누설 경로를 검증하는 계획을 세웠습니다.

PROCESS REALITY

Geometry–capacitance trade-off

교수 피드백을 반영해 MEB depth와 overlap capacitance 상관을 분석 범위에 추가했습니다.

03 / WORKFLOW

Planned TCAD workflow.

  1. Baseline structure

    20 nm급 BCAT geometry와 off-state bias를 정의하고 trench corner mesh를 세분화합니다.

  2. Model validation

    BandToBand 모델, 380 K 조건, GIDL 전류와 전계 분포의 추출 방식을 점검합니다.

  3. Phase 1 sweep

    trench depth, LDD, Halo 조건을 변경해 민감도와 물리적 방향성을 비교합니다.

  4. Phase 2 refinement

    MEB depth와 gate–drain overlap, Cov의 상관관계를 포함해 후보 조건을 정교화합니다.

04 / ANALYSIS

From first proposal to refined scope.

초기 제안은 trench depth와 LDD/Halo 조건을 중심으로 GIDL을 낮추는 방향이었습니다. 재발표 자료에서는 “수치가 좋아지는 변수”를 찾는 데 그치지 않고, 실제 BCAT 구조의 MEB 공정과 gate overlap이 만드는 capacitance까지 함께 보라는 피드백을 반영해 두 단계 최적화 계획으로 확장했습니다.

07 / LIMITS

Limitations & lessons.

  • 프로젝트는 진행 중이며 공개 저장소에는 최종 TCAD 결과, 수치 비교표, 검증된 최적 recipe가 없습니다.
  • README와 발표 PDF 외에 command deck이나 simulation result 파일이 없어 코드 뷰어를 제공하지 않습니다.
  • GIDL 감소만으로 설계를 평가할 수 없고, MEB geometry와 Cov 등 구조·동작 trade-off를 함께 다뤄야 합니다.
  • 향후 결과는 사용한 BTBT 모델, mesh, 바이어스 조건에 대한 민감도와 함께 제시되어야 합니다.
