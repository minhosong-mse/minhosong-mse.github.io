SEMICONDUCTOR PROCESS · SENTARUS TCAD

NMOS & PMOS Process Optimization

공정 변수 탐색부터 극성 변환, 전기적 지표 추출까지

교과목에서 제공한 SimpleMOS 흐름을 바탕으로 NMOS 성능을 최적화하고, 동일 공정의 도핑 극성과 바이어스를 PMOS에 맞게 변환한 개인 프로젝트입니다. Sprocess–Sdevice–Svisual 흐름을 연결해 공정 조건과 Ion, Ioff, SS, Vtgm의 관계를 추적했습니다.

01 / OVERVIEW

Two connected optimization tasks.

첫 단계에서는 NMOS 기준 공정에서 gate length, 산화 시간, 기판 농도, LDD·source/drain implant 조건을 탐색해 성능 균형점을 찾았습니다. 이어서 기판, LDD, source/drain의 도핑 종과 게이트 바이어스를 PMOS 동작에 맞게 전환하고 5개 변수를 다시 최적화했습니다.

NMOS TARGETIon/Ioff 확대와 SS 유지
PMOS TARGET극성 변환 후 안정적 transfer 특성 확보
PIPELINESprocess → Sdevice → Svisual
EVIDENCEREADME · 31-page report · result images
검증된 성과: 최종 NMOS 조건은 교과목 내 최적화 콘테스트 공동 1위로 기록되어 있습니다.

02 / PROBLEM

A multi-objective process problem.

Ion을 키우는 조건이 항상 Ioff와 SS에 유리하지 않으며, implant dose·energy와 열처리 시간은 접합 깊이와 도핑 분포를 함께 바꿉니다. 따라서 단일 최대값이 아니라 누설, 구동 전류, 문턱 특성을 동시에 비교할 평가 기준이 필요했습니다.

NMOS

넓은 탐색에서 세분화로

초기 범위에서 유망한 조합을 찾고, NWell 농도와 LDD dose 구간을 세분화해 최종 조건을 선정했습니다.

PMOS

공정 변환 뒤 재최적화

Phosphorus N-well, Boron LDD·source/drain, 음의 gate sweep을 적용한 뒤 implant와 RTA의 영향을 비교했습니다.

03 / WORKFLOW

Process-to-device workflow.

  1. Baseline & metrics

    기준 구조와 transfer curve를 확인하고 Ion/Ioff, SS, Vtgm, gm을 비교 지표로 설정했습니다.

  2. NMOS parameter search

    약 1,000개 이상의 조합을 탐색한 보고서 기록을 바탕으로 유망 범위를 좁혔습니다.

  3. PMOS conversion

    기판·LDD·source/drain 도핑과 gate sweep의 극성을 PMOS에 맞게 변경했습니다.

  4. Extraction & review

    Svisual에서 전기적 지표를 추출하고 공정 구조·Id–Vg 결과를 함께 검토했습니다.

04 / ANALYSIS

Implementation and interpretation.

Sprocess에서 공정 변수를 Workbench 파라미터로 노출하고 주요 단계마다 TDR 구조를 저장했습니다. Sdevice는 PMOS에서 drain 전압을 설정한 뒤 gate를 0 V에서 −2.5 V까지 스윕했으며, Svisual은 곡선으로부터 Vtgm, 전류 극값, Ioff, SS, gm을 추출했습니다.

06 / CONTRIBUTION

My contribution.

  • 개인 프로젝트로 NMOS 후보 조건 탐색, PMOS 공정 변환, 변수 split, 결과 해석 전 과정을 수행했습니다.
  • Sprocess에 PMOS 도핑 종과 5개 최적화 파라미터를 반영하고 공정 단계별 구조 출력을 구성했습니다.
  • Sdevice의 음전압 gate sweep과 Svisual의 전기적 지표 추출 흐름을 정리했습니다.
  • 공정 조건–도핑 분포–transfer 특성의 연결 관계를 보고서와 저장소 문서로 구조화했습니다.

07 / LIMITS

Limitations & lessons.

  • NMOS의 원본 command 파일은 저장소에 보존되어 있지 않으며, NMOS 수치와 과정은 README와 보고서에 남은 결과를 근거로 정리했습니다.
  • PMOS 코드는 보고서와 Workbench 작업 내용을 바탕으로 README에 정리되어 있고, 독립 실행형 command 파일은 저장소에 없습니다.
  • 결과는 사용한 TCAD 물리 모델과 경계조건에 의존하며, 실측 소자와의 검증을 포함하지 않습니다.
  • 단일 지표의 극대화보다 Ion, Ioff, SS, Vtgm의 trade-off를 함께 판단해야 공정 선택이 물리적으로 일관됨을 확인했습니다.
